MN Global BTC Treasury Dashboard

An institutional-grade command center for MN Global Investment Holdings LLC. This interface provides real-time monitoring of Bitcoin treasury assets, tracking total holdings, cost basis, and market valuation.

Core Features:

    Treasury Analytics: Real-time USD valuation and PnL tracking.

    Wallet Oversight: Monitoring for Coinbase, ECOS, and segregated cold storage.

    Security Protocols: Sentinel-grade monitoring for dormant long-term reserves.

    Policy Compliance: Automated tracking of internal spend limits and multi-sig requirements.
